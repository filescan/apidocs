Our API

Welcome to FileScan API documentation!

Base URL for our API is https://api.filescan.dev/v1/

API Keys

This is API keys documentation. You can use this API to request for details on API keys.

Check API calls limit

Check API calls limit left

/v1/api_keys

$ curl https://api.filescan.dev/v1/api_keys

Example response

{
    "data": {
        "attributes": {
            "limit_left": 9,
            "secret_key": "sk_fcca1eb92d8452962fcf2d8905f4f5ef"

        "relationships": {
            "project": {
                "data": {
                    "id": "1",
                    "type": "project"
                }
            },
            "user": {

                }

    }
}

Projects

This is projects API documentation. You can use this API to request for details and remove different projects.

List projects

Lists all scans for a particular account.

GET
/v1/projects

Example request

$ curl https://api.filescan.dev/v1/projects

Example response

{
    "data": [
        {
            "attributes": {
                "api_limit": 20,
                "name": "test",
                "usage_count": 11
            },
            "id": "1",
            "relationships": {
                "user": {
                    "data": {
                        "id": "1",
                        "type": "user"
                    }
                }
            },
            "type": "project"
        }
    ]
}

Retrieve information about an existing project.

            "user": {
                "data": {

}

Update a project

Updates the properties of a particular project.

Property Description
name (optional) the name of the project
PATCH
/v1/projects/{project_id}

Example request

curl --request PATCH https://api.filescan.dev/v1/projects/{project_id} \
  -d @data.json

Example request body

{
  "name": "foo",
}

Example response

{
    "data": {
        "attributes": {
            "api_limit": 20,
            "name": "foo",
            "usage_count": 11
        },
        "id": "1",
        "relationships": {
            "user": {
                "data": {
                    "id": "1",
                    "type": "user"
                }
            }
        },
        "type": "project"
    }
}

IMPORTANT: All scans are assigned to Projects, so you must pass project id when doing request

Create a scan

Scans the uploaded file

POST
/v1/projects/{project_id}/scans

Example request

curl -X POST -F "file=@file" https://api.filescan.dev/v1/{project_id}/scans

Example response

{
  "data": {
    "id": "19",
    "type": "scan",
    "attributes": {
      "filename": "Makefile",
      "status": null,
      "scan_details": null,
      "created_at": "2019-07-15T07:37:07.877Z"
    },
    "relationships": {
      "user": {
        "data": {
          "id": "1",
          "type": "user"
        }
      },
      "project": {
        "data": {
          "id": "1",
          "type": "project"
        }
      }
    }
  }
}
