Our API

Welcome to FileScan API documentation!

Base URL for our API is https://api.filescan.dev/v1/

API Keys

This is API keys documentation. You can use this API to request for details on API keys.

Projects

This is projects API documentation. You can use this API to request for details and remove different projects.

List projects

Lists all scans for a particular account.

GET
/v1/projects

Example request

$ curl https://api.filescan.dev/v1/projects

Example response

{
    "data": [
        {
            "attributes": {
                "api_limit": 20,
                "name": "test",
                "usage_count": 11
            },
            "id": "1",
            "relationships": {
                "user": {
                    "data": {
                        "id": "1",
                        "type": "user"
                    }
                }
            },
            "type": "project"
        }
    ]
}

Retrieve a project

Returns a single project.

Retrieve information about an existing project.

GET
/v1/projects/{project_id}

Example request

curl https://api.filescan.dev/v1/{project_id}

Example response

    "data": {
        "attributes": {
            "api_limit": 20,
            "name": "test",
            "usage_count": 11
        },
        "id": "1",
        "relationships": {
            "user": {
                "data": {
                    "id": "1",
                    "type": "user"
                }
            }
        },
        "type": "project"
    }
}

Update a project

Updates the properties of a particular project.

Property Description
name (optional) the name of the project
PATCH
/v1/projects/{project_id}

Example request

curl --request PATCH https://api.filescan.dev/v1/projects/{project_id} \
  -d @data.json

Example request body

{
  "name": "foo",
}

Example response

{
    "data": {
        "attributes": {
            "api_limit": 20,
            "name": "foo",
            "usage_count": 11
        },
        "id": "1",
        "relationships": {
            "user": {
                "data": {
                    "id": "1",
                    "type": "user"
                }
            }
        },
        "type": "project"
    }
}

List scans

List all the scans in a project. The response body will be a ScanCollection.

GET
/v1/projects/{project_id}/scans

Example request

curl https://api.filescan.dev/v1/projects/{project_id}/scans

Example response

{
    "data": [

                "created_at": "2019-07-12T09:24:15.475Z",
